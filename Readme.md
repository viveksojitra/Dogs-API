## 📦 Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/viveksojitra/Dogs-API.git
    ```
2. Navigate to the project directory:
    ```bash
    cd blogger-admin
    ```
3. Start the server:
    ```bash
    start index.html
    ```
